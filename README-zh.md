## 🧞 命令

所有命令都从项目的根目录的终端运行：

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | 安装依赖项                            |
| `npm run dev`             | 启动本地开发服务器 `localhost:4321`      |
| `npm run build`           | 构建您的生产站点 `./dist/`          |
| `npm run preview`         | 部署之前在本地预览您的构建    |
| `npm run astro ...`       | 运行 CLI 命令，例如 `astro add`, `astro check` |
| `npm run astro -- --help` | 使用 Astro CLI 获取帮助                     |
